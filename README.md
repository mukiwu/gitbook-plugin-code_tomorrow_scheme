# Tomorrow color scheme for GitBook Code


## how to install

Step1. save to package.json

```
npm install gitbook-plugin-code_tomorrow_scheme --save
```

Step2. config book.json

```
{
    "plugins": ["code_tomorrow_scheme"]
}
```

Setp3. build

```
gitbook build ./
```
